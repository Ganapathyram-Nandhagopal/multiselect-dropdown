# 🎯 MultiSelect Dropdown

A modern animated multi-select dropdown component built with HTML, CSS/SCSS, JavaScript, and jQuery. Features smooth tag animations, ripple effects, dropdown transitions, dynamic item selection, and a clean user interface suitable for ERP systems, dashboards, SaaS applications, and modern web projects.

---

## ✨ Features

* 🚀 Pure Frontend Component
* 🎭 Smooth Add & Remove Animations
* 🎯 Multi-Selection Support
* ⚡ Dynamic Tag Creation
* 💫 Ripple Click Effects
* 📂 Animated Dropdown Menu
* 🎨 Modern UI Design
* 📱 Responsive Layout
* ♿ Accessibility-Friendly
* 🔧 Easy Customization
* 🖱️ Keyboard & Mouse Friendly
* 🌐 GitHub Pages Compatible

---

## 🛠️ Technologies Used

* HTML5
* CSS3
* SCSS
* JavaScript (ES6)
* jQuery

---

## 🎬 Animations Included

| Animation      | Description                                  |
| -------------- | -------------------------------------------- |
| Tag Creation   | Smooth tag appearance when selecting an item |
| Tag Removal    | Animated tag deletion effect                 |
| Ripple Effect  | Interactive ripple animation on selection    |
| Dropdown Scale | Smooth dropdown open/close transition        |
| Fade Effects   | Elegant fade-in and fade-out interactions    |
| Arrow Rotation | Dropdown arrow rotates during toggle         |
| Chip Expansion | Selected items animate into view             |
| Hover Effects  | Interactive hover feedback on options        |
